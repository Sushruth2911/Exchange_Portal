# Exchange_Portal

Exchange Planning website

Getting Started
Prerequisites
Make sure you have the following installed on your machine:

Node.js v18 or higher
npm v9 or higher (comes with Node.js)
You can verify your versions by running:

node -v
npm -v
Installation
Clone or download the repository and navigate into the project folder:

cd ExchangePlanner
Install all dependencies (React, Tailwind CSS, React Router, Lucide icons, and TypeScript tooling):

npm install
This will install everything listed in package.json — both the runtime dependencies and dev tools.

Running the App
Start the local development server:

npm run dev
Then open your browser and go to:

http://localhost:5173
The app will hot-reload automatically whenever you save a file.

Building for Production
To compile TypeScript and bundle the app for deployment:

npm run build
The output will be in the dist/ folder. To preview the production build locally:

npm run preview
